Functions

Creating your own function

Programming with R: Functions

You can create your own function so you can use it multiple times

my_function <- function(){}

For example, we need to create a function that generates 10 numbers between 1 and 100 y returns the sum of all of them.

Code 
gen2_1_100 <- function(){
  nums <- sample(1:100, size = 10, replace = TRUE)
  print(nums)
  return(sum(nums))
}

Functions

Creating your own function

Programming with R: Functions

Once created you can use it…

x <- gen2_1_100()
 [1]  5 74 18 25 40 15 17 44 74 46
x
[1] 358

Now modify the function to have an argument that tells the maximum numbers to generate, but with a default value of 2.

Code 
# we add a paremeter for the size with a default value of 2
gen2_1_100 <- function(n = 2){
  nums <- sample(1:100, size = n, replace = TRUE)
  print(nums)
  return(sum(nums))
}

Packages & Libraries

Packages & Libraries

Programming with R: Packages & Libraries

As we have seen, R base has a lot of pre-defined functions and tools you can use when programming. But sometimes (very often) we will need more tools don’t available by default… we can build a function to do that… but probably we can use already created packages by other users that have a lot of functions that do what we need.

A package is an extension for R that includes functions and data sets we can use and load in our projects. The possibility of using and creating code packages makes powerful R helping doing different tasks.

You can learn more about packages here.

Packages & Libraries

Programming with R: Packages & Libraries

Installing and loading a package

For installing a package we just need to type the following command in our R console…

install.packages("tidyverse")

When installing packages with this command, never type it on the script to avoid issues and possible problems if you already have the library installed.

Once installed we need to load it to use it, we can do that writing in our scripts the following command at the beginning of the code…

library("tidyverse")

It is recommended to write this commands at the beginning of the script so the programmer can see all libraries needed.

Programming Structures

Programming Structures

Programming structures are essential to control and manage your code flow, the two main programming structures used are…

  • Conditional structures. Allows us to control if a certain piece of code needs to be executed depending on a logical condition.

  • Iterative structures (loops). Allows us to perform repetitive tasks with less code.

Conditional Structures

Will control which parts of the code will be executed depending if the condition is fulfilled.

if(condition){
 do this if the condition is TRUE
}

You can combine them…

x <- 3
if (x > 2) { 
  if (x < 5) {
    print("x is greater than 2")
  }
}
[1] "x is greater than 2"

Try to do it with a single condition!

Conditional Structures

You can manage alternative path for your code with else statement.

if(condition){
 do this if the condition is TRUE
} else {
 do this if the condition is FALSE
}

And you can manage more conditions with else if

if(condition 1){
 do this if the condition 1 is TRUE
} else if (condition 2) {
 do this if the condition 2 is TRUE
} else {
 do this if none of the conditions are TRUE
}

Conditional Structures

What will happen with the following code…

a <- 10
b <- -3

# %% operator will return the module of a division
# we check if the value is even
if(a %% 2 == 0 & b %% 2 == 0){ 
  c <- a / 2 - b / 2
} else if(a %% 2 & b < 0){
  c <- a / 2 - b
} else {
  c <- a + b
}
c

Loops

Repeat operations multiple times. There are several structures here, we will focus on two of them:

  • for loops
  • while loops

Loops: for

Repeat the code several times, depending on the number of elements in the given range or collection.

for (value in collection){
  do this
}

Some examples…

for(x in c("A", "B", "C")){
  print("Iteration")
}
[1] "Iteration"
[1] "Iteration"
[1] "Iteration"
for(y in c("A", "B", "C")){
  print(y)
}
[1] "A"
[1] "B"
[1] "C"

Loops: while

Repeat the code several times, while the condition typed is fulfilled.

while (condition){
  do this
}

An example…

x <- 1
while(x < 8){
  print(x)
  x <- x + 1
}
[1] 1
[1] 2
[1] 3
[1] 4
[1] 5
[1] 6
[1] 7
